# 🚀 Welcome to The WFA!

## About Us
The Wheelchair Football Association is the governing body for Powerchair Football in England. We develop many systems to improve internal workflows and also external engagement with the amazing sport of Powerchair Football.

## 📌 Projects
We are actively working on exciting projects, including:
- ⚽️ **WFA MatchDay** - A database system which contains all data about Powerchair Football matches, players and teams.
- 📱 **[WFA ScoreLine](https://scoreline.thewfa.org.uk)** - A mobile application which enables the consumtion of Powerchair Football statistics.
- 🤖 **WFAi** - A collection of AI tools to improve understanding of the game and generate more in-depth statistics.
